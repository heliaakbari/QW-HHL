# QW-HHL
A quantum-walk-modified HHL procedure

## Article
C. D. Phillips, H. Akbari, and V.I. Okhmatovski, ``A Quantum Computer Amenable Sparse Matrix Equation Solver,'' arXiv:2112.02600 [quant-ph], 2022. [Online]. Available: https://arxiv.org/abs/2112.02600

## Important Files and Folders Guide:

  ✅ requirements.txt : Python libraries dependancy list.
  
  ✅ Hardware_Run : Files to run on IBM runtime on real quantum backend.
  
  ✅ Main2Unknown.py : Simplified circuit for the 2x2 trivial case.
  
  ✅ Main4Unknown.py : Simplified circuit for the 4x4 trivial case.
  
  ✅ Software_Simulation : Simulation code for arbitrary matrix.
  
  ✅ matrix_dat : Files that have been utilized as random test cases.
  
  ✅ MatrixProcedures.py : Prepare the system from input matrix, compare with classical solution, and print matrix.
  
  ✅ QAlgs.py : Implementation of quantum algorithms such as HHL rotation, QPE, and QFT.
  
  ✅ QWOps.py : Implementation of quantum operations of the paper such as T, Bj, and W.
  
  ✅ Old_Version : Initial code of the paper that is compatible with older version of Qiskit.
